# Portafolio React

Portafolio personal desarrollado con React y Vite.

## Instalación

```bash
git clone https://github.com/Torres20939/Portafolio-React.git
cd Portafolio-React
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

El sitio estará listo en la carpeta `dist/`.

## Despliegue

Simplemente sube a GitHub. Puedes desplegar en:
- **Vercel**: Conecta tu repo en vercel.com
- **Netlify**: Conecta tu repo en netlify.com  
- **GitHub Pages**: Habilita en Settings → Pages

## Tecnologías

- React 19
- Vite 7
- React Router 7
- Lucide React
